# Open Techiz Clone

A high-fidelity, production-ready clone of the [Open Techiz](https://opentechiz.com) corporate website. This website was built solely for educational and learning purposes. The original design, concept, and content belong to [Open Techiz](https://opentechiz.com).

## Built With

*   **Next.js 16 (App Router)** - React framework for production
*   **React 19** - UI library
*   **Tailwind CSS v4** - Utility-first CSS framework for custom styling
*   **TypeScript** - Static typing for scalable development
